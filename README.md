AI Travel Planner is a modern, intelligent travel companion that takes the hassle out of trip planning. Powered by Google Gemini AI, it generates personalized day-by-day itineraries based on your preferences, budget, and travel style.

---

## ✨ Key Features

- 🤖 **AI-Driven Itineraries**: Get custom travel plans generated by Google Gemini AI.
- 📍 **Smart Destination Search**: Integration with Google Places API for accurate location suggestions.
- 💰 **Budget Management**: Choose between 'Cheap', 'Moderate', or 'Luxury' to fit your spending.
- 👥 **Traveler Customization**: Tailor the trip for 'Solo', 'Couple', 'Family', or 'Friends'.
- 🏨 **Hotel Recommendations**: Discover the best places to stay with detailed info.
- 🗺️ **Interactive Itinerary**: View a structured, day-wise plan of places to visit.
- 🔐 **Secure Google Auth**: Quick and easy login using Google OAuth.
- 📂 **Save & Manage Trips**: All your generated trips are securely stored in Firebase Firestore.

## 🚀 Tech Stack

- **Frontend**: React.js, Vite, Tailwind CSS
- **UI Components**: Shadcn UI, Lucide React, React Icons
- **AI Engine**: Google Generative AI (Gemini Pro)
- **Backend / DB**: Firebase Firestore
- **Maps & Places**: Google Maps Platform (Places API)
- **Authentication**: Google OAuth 2.0
- **HTTP Client**: Axios

## 🛠️ Getting Started

### Prerequisites

- Node.js installed on your machine.
- A Google Cloud Project (for Maps & Gemini API).
- A Firebase Project (for Firestore).

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/AI-Travel-Planner.git
   cd AI-Travel-Planner
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Environment Variables:**
   Create a `.env.local` file in the root directory and add the following:
   ```env
   VITE_GOOGLE_PLACE_API_KEY=YOUR_GOOGLE_MAPS_API_KEY
   VITE_GOOGLE_GEMINI_AI_API_KEY=YOUR_GEMINI_API_KEY
   VITE_GOOGLE_AUTH_CLIENT_ID=YOUR_GOOGLE_CLIENT_ID
   ```

4. **Firebase Configuration:**
   Update the `src/service/firebaseConfig.jsx` file with your own Firebase project credentials.

5. **Run the development server:**
   ```bash
   npm run dev
   ```
